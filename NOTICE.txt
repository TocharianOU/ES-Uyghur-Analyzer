This project uses the THUUyMorph dataset, which is a morphology-based Uyghur language segmentation corpus provided by Tsinghua University Natural Language Processing and Computational Social Science Lab.

More information and the dataset can be found at: http://thuuymorph.thunlp.org/

If you use this dataset in your work, please cite the following paper:
THUUyMorph-A Uyghur Morphological Analysis Corpus. In Proceedings of CCL/NLP-NABD 2017, Nanjing, October.

The dictionaries `ug_mor_original.txt` and `ug_mor_split.txt` used in this project are also based on the resources provided by Tsinghua University and the mentioned paper.
